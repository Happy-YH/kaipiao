================================================================
  开票系统（kaipiao / invoicing-system-design）日常操作命令备忘
  分支：trae/agent-Pm4iTC
================================================================


----------------------------------------------------------------
1. 每天开始工作前（任一云端）
----------------------------------------------------------------
cd /workspace/kaipiao          # 或 /workspace/invoicing-system-design
git pull origin trae/agent-Pm4iTC


----------------------------------------------------------------
2. 修改代码后（提交并推送）
----------------------------------------------------------------
git add .
git commit -m "feat: 描述你的修改"
git push origin trae/agent-Pm4iTC


----------------------------------------------------------------
3. 在另一台云端拉取最新代码
----------------------------------------------------------------
git pull origin trae/agent-Pm4iTC


----------------------------------------------------------------
4. 重新构建并启动后端
----------------------------------------------------------------
cd backend
mvn package -DskipTests
pkill -f invoice-system
nohup java -jar target/invoice-system-1.0.0.jar > /tmp/backend.log 2>&1 &


----------------------------------------------------------------
5. 重新构建并启动前端
----------------------------------------------------------------
cd ../frontend
pkill -f vue-cli-service
nohup npm run dev > /tmp/frontend.log 2>&1 &


----------------------------------------------------------------
附：一行串起来执行的写法（构建后端 + 启动后端）
----------------------------------------------------------------
cd backend && mvn package -DskipTests && \
  pkill -f invoice-system; nohup java -jar target/invoice-system-1.0.0.jar > /tmp/backend.log 2>&1 &


----------------------------------------------------------------
附：一行串起来执行的写法（启动前端）
----------------------------------------------------------------
cd ../frontend && \
  pkill -f vue-cli-service; nohup npm run dev > /tmp/frontend.log 2>&1 &


----------------------------------------------------------------
查看日志
----------------------------------------------------------------
tail -f /tmp/backend.log     # 后端日志
tail -f /tmp/frontend.log    # 前端日志


----------------------------------------------------------------
停止服务
----------------------------------------------------------------
pkill -f invoice-system       # 停后端
pkill -f vue-cli-service      # 停前端


================================================================
提示：
- 后端 jar 名称以 backend/target 下实际产物为准（默认 invoice-system-1.0.0.jar）。
- 前端默认端口见 frontend/vue.config.js，启动后访问对应地址。
- 提交前请确认是否需要 .gitignore 排除 target/、node_modules/ 等。
================================================================
